1. 运行登录命令。

    ```azurecli-interactive
    az login
    ```

   将获得要在下一步骤中使用的代码。

1. 使用 Web 浏览器打开页面 [https://aka.ms/devicelogin](https://aka.ms/devicelogin)，输入该代码进行身份验证。

    系统会提示使用凭据登录。

1. 登录。
