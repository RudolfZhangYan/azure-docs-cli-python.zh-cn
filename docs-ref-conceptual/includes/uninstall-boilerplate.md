如果你决定卸载 Azure CLI，我们会很遗憾。 在卸载之前，请使用 `az feedback` 命令告诉我们在哪些方面需要改进或修复。 我们的目标是消除 Azure CLI 中的 bug 并使其变得用户友好。 如果你在发现 bug 后[提出 GitHub 问题](https://github.com/Azure/azure-cli/issues)，我们将十分感激。
