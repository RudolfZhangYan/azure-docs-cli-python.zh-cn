1. 运行 `login` 命令。

    ```azurecli-interactive
    az login
    ```

    如果 CLI 可以确定默认浏览器并且有权打开它，它将打开它并立即将你引导到登录页。

    否则，你便需要打开一个浏览器页面，并在浏览器中导航到 [https://aka.ms/devicelogin](https://aka.ms/devicelogin) 后按照有关命令行的说明输入授权代码。

2. 在浏览器中使用帐户凭据登录。
