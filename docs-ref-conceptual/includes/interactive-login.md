1. 运行 `login` 命令。

    ```azurecli-interactive
    az login
    ```

    如果 CLI 可以打开默认的浏览器，则它会打开该浏览器并加载登录页。

    否则，你便需要打开一个浏览器页面，并在浏览器中导航到 [https://aka.ms/devicelogin](https://aka.ms/devicelogin) 后按照有关命令行的说明输入授权代码。

2. 在浏览器中使用帐户凭据登录。
